# Backend проекта GrandTrip — портала для туристов с авторскими маршрутами по Санкт-Петербургу

## Функционал
- Создание и детальное редактирование маршрутов:
  - добавление и подробное описание точек
  - автоматическое прокладывание маршрутов между двумя точками с использованием OpenStreetMap API
- Фильтрация маршрутов по теме, сезону, длительности
- Админ-панель для управления пользователями и маршрутами

## Стек технологий

- **C# / ASP.NET Core**
- **Entity Framework Core**
- **MSSQL Server**

## Проект создавался в учебных целях
Некоторые части проекта нуждаются в рефакторинге. Необходимо:
- Реорганизовать слои логики в разных частях приложения
- Привести все эндпоинты к единой REST-архитектуре
- Проработать валидацию, обработку ошибок и стандартные ответы
